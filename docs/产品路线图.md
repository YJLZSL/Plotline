# Plotline 产品路线图

> 本文档记录 Plotline 的已发布版本、当前开发计划与未来候选方向。

> ---

## 当前版本

### v3.2.0（开发中：时间轴拖动吸附）
- 时间轴拖动吸附：新增 `getSnapTimeAtX` / `getSnapXAtTime` 纯函数，基于当前标尺档位将事件拖动吸附到最近的时间网格
- 吸附提示组件：`DragSnapHint` / `DragSnapTooltip` 实时显示吸附目标时间与偏移量
- EventCard 拖动归位动画：释放后沿 `motionOrchestrator` 的 `dragSnap` 预设播放归位动画，连接线同步跟随锚点
- 动画编排层扩展：`motionOrchestrator` 新增 `dragSnap` 场景预设，统一管理吸附确认、卡片归位、连接线重绘
- 真实创作流程 E2E：覆盖创建事件 → 拖动吸附 → 验证时间与视觉位置一致
- 版本号统一：`package.json`、`Cargo.toml`、`tauri.conf.json`、`Cargo.lock` 升级到 `3.2.0`
- GitHub Release: 待发布

## 已发布版本

### v3.1.0（已发布：时间轴对齐重写与动画编排层）
- 时间轴坐标单一源重写：新增 `ViewportState` + `getXAtTime` / `getTimeAtX` 纯函数，统一标尺、事件卡片、连接线、Today 线坐标计算
- 时间轴标尺 6 档刻度自适应：`chooseTickLevel(zoom)` 在 year / quarter / month / week / day / hour 之间切换
- 事件卡片三 zone 重构：header（状态点+标题+连接句柄）/ body（时间范围+持续时间）/ footer（地点+角色头像堆叠）
- 连接线时间锚点边缘对齐：同轨道左右沿中点 / 跨轨道上下沿中点，拖动事件后始终贴合锚点
- 动画编排层 `motionOrchestrator`：5 个场景预设（viewSwitch / cardBatchEnter / dragReturnWithConnections / aiPanelExpand / sidebarNavEnter）+ 分层缓动 + `prefers-reduced-motion` 退化
- 工具栏四分组（Create / Filter / View mode / More）+ Sidebar 三分组（工作区视图 / 创作辅助 / 系统）
- 首次进入工作区高亮"时间轴"与"AI 创作"入口；删除重复的"AI Studio"入口
- 14 个 visual E2E 测试（含 dev server 不可用时自动 skip 机制）
- 版本号统一：`package.json`、`Cargo.toml`、`tauri.conf.json`、`Cargo.lock` 升级到 `3.1.0`
- GitHub Release: <https://github.com/YJLZSL/Plotline/releases/tag/v3.1.0>

### v3.0.0（已发布）
- 时间轴/事件逻辑重写：修复拖拽锚点、事件重叠回归、相对/绝对事件混合布局，统一缩放与平移视口
- Timeline / Script 双视图双向同步：新增独立剧本视图，按故事时间线阅读事件并跳转回时间轴选中
- 新增 `workspaceSelection` 全局选择状态，跨视图共享 `selectedEventId`
- 新增 AI 创作助手侧边模块：7 个 Agent（对话/续写/脑暴/查漏/润色/角色关系/文风迁移）、会话管理、上下文选择器

- 测试覆盖：新增 Timeline 平移缩放、Script 同步、AI 助手 E2E 测试；新增 `scriptSort` 单元测试
- 版本号统一：`package.json`、`Cargo.toml`、`tauri.conf.json`、`Cargo.lock` 升级到 `3.0.0`
- GitHub Release: <https://github.com/YJLZSL/Plotline/releases/tag/v3.0.0>

## 当前开发中：v3.2+

## v3.2+ 候选方向

### 候选 1：地图增强
- 地点分组 / 图层开关
- 地点自定义图标与角色足迹连线
- 地图打印 / PDF / 图片导出

### 候选 2：VN 编辑器增强
- 角色立绘插槽拖拽定位
- 完整预览播放器（带分支选择）
- Ren'Py 导出增强（音频、转场、变量）

### 候选 3：世界观实体
- 种族 / 物种 / 宗教 / 职业等实体管理
- 实体与角色、地点、事件关联
- 冲突检测：发现设定矛盾

### 候选 4：AI Agent 工作流
- 自动整理大纲、发现时间轴漏洞
- 角色关系网络一致性检查
- 长上下文 RAG：基于当前工作区全部创作生成建议
- AI 输出版本回溯

### 候选 5：UI 美术与国际化
- 统一空状态插画（已部分落地，继续补全各视图）
- 卡片质感升级（玻璃/纸质拟物）
- 日文翻译补全

### 候选 6：番茄钟统计
- 与写作目标绑定
- 每日/每周专注统计
- 系统通知提醒

## 决策记录

- MC 元素采用原创像素风 SVG，避免直接使用 Mojang/Microsoft 版权素材。
- AI 厂商图标优先使用官方/品牌 SVG 资源；品牌图标版权归各公司所有。
- 新功能必须随代码提交同步更新 `AGENTS.md`、`更新日志.md`、测试与路线图。
