# Plotline 产品路线图

> 本文档记录 Plotline 的已发布版本、当前开发计划与未来候选方向。

---

## 已发布版本

### v2.2.0（已发布）
- 时间轴真实日历映射：新增 `timeScale.ts` 替换固定 30/365 近似，`TimelineView` 与 `DateRuler` 严格对齐
- 事件拖拽重写：移除 `dragConstraints`，支持真实横向拖拽、相邻事件排序、连接点击删除
- 一致性冲突可视化：事件卡片红色角标/边框，编辑对话框顶部提示
- 事件颜色选择器：马卡龙色板
- 动画性能优化：统一 150-200ms 动效、`prefers-reduced-motion`、CSS active 反馈、全局保存状态指示、`beforeunload` 保护
- MC 主题再设计：泥土棕/草绿/红石红/圆石灰/木板褐配色，控件方块化
- 番茄钟音效闭环：阶段切换/完成均播放音效，MC 主题完成播放 explosion
- 工作区保存改进：元数据编辑、定时备份调度器、undo/redo 扩展至工作区、Bundle 版本校验、子实体变更联动 `updated_at`
- 自定义字体导入：默认内置得意黑（Smiley Sans），支持导入 `.ttf/.otf/.woff/.woff2` 到 `app_data/fonts/`
- GitHub Release: <https://github.com/YJLZSL/Plotline/releases/tag/v2.2.0>

### v2.1.0（已发布）
- 时间轴可读性升级：主次刻度、今天参考线、轨道 lane 交替背景、事件卡片状态色
- 视图切换动画：统一 `WorkspaceLayout` 路由切换，修复元素重复与抖动
- MC 全局主题：草方块绿配色、像素字体、按钮/卡片/输入框方块风格
- MC 彩蛋：番茄钟心形/苦力怕脸、红石块/草方块图标、`SplashOverlay` 草方块入场动画、Web Audio 合成 MC 风格音效
- AI 真实厂商图标：从 LobeHub 品牌 SVG 抓取官方/高辨识度图标
- 按功能 AI 助手：`aiContext` 共享创作上下文，时间轴/角色/大纲/地图/VN/世界观/资料库均有建议/改写入口
- 统计新图表：思维导图（径向树）、脑状图（角色关系网络）、树状图（大纲层级）
- 真实浏览器测试：Playwright 截图覆盖时间轴、MC 主题、AI 面板、统计图表，现场修复切换动画重复问题
- GitHub Release: <https://github.com/YJLZSL/Plotline/releases/tag/v2.1.0>

### v2.0.0（已发布）
- 世界观模块（历史/魔法/势力/地理/文化/其他）
- 工作区导出：PDF / Word / EPUB
- AI 流式输出 + 可配置系统提示词
- VN 富文本台词与分支一致性检查
- WorkspaceBundle 补全（地点、连线、VN 场景/台词）
- CSP 白名单
- GitHub Release: <https://github.com/YJLZSL/Plotline/releases/tag/v2.0.0>

## 当前开发中：v2.3+

### 目标
基于 v2.2.0 的反馈，继续增强地图、VN、世界观、AI Agent 与 UI 美术，并补全番茄钟统计。

---

## v2.3+ 候选方向

### 候选 1：地图增强
- 地点分组 / 图层开关
- 地点自定义图标与角色足迹连线
- 地图打印 / PDF / 图片导出

### 候选 2：VN 编辑器增强
- 角色立绘插槽拖拽定位
- 完整预览播放器（带分支选择）
- Ren'Py 导出增强（音频、转场、变量）

### 候选 3：世界观实体
- 种族 / 物种 / 宗教 / 职业等实体管理
- 实体与角色、地点、事件关联
- 冲突检测：发现设定矛盾

### 候选 4：AI Agent 工作流
- 自动整理大纲、发现时间轴漏洞
- 角色关系网络一致性检查
- 长上下文 RAG：基于当前工作区全部创作生成建议

### 候选 5：UI 美术与国际化
- 统一空状态插画
- 卡片质感升级（玻璃/纸质拟物）
- 日文翻译补全

### 候选 6：番茄钟统计
- 与写作目标绑定
- 每日/每周专注统计
- 系统通知提醒

---

## 决策记录

- MC 元素采用原创像素风 SVG，避免直接使用 Mojang/Microsoft 版权素材。
- AI 厂商图标优先使用官方/品牌 SVG 资源；品牌图标版权归各公司所有。
- 新功能必须随代码提交同步更新 `AGENTS.md`、`更新日志.md`、测试与路线图。
