# Plotline PRD 实现度审计报告（v2.5.4）

> 审计日期：2026-06-26  
> 审计对象：`产品需求与设计文档.md`（PRD v1.1.0） vs. 当前代码库（v2.5.4）  
> 审计范围：核心功能模块 2.1–2.7 及非功能性需求 7.x  
> 评分标准：10=完全实现，7=基本实现/小缺陷，4=部分实现/明显缺口，0=未实现

---

## 一、总体结论

| 指标 | 数值 |
|---|---|
| 总体评分 | **7.5 / 10** |
| 核心工作流可用性 | 高 |
| PRD 覆盖度 | 约 75% |
| 超出 PRD 的附加价值 | 高（地图、VN、小说、AI、番茄钟等） |

**一句话总结**：时间轴、角色、大纲、统计等核心叙事创作闭环已经跑通，交互与视觉完成度较高；但 PRD 中强调的“全局自动保存”“富文本描述”“大纲拖拽排序”“每日写作目标”“快捷键自定义”“高对比度/全键盘可访问性”等关键项仍未落地，是 v2.6.0 最值得补齐的缺口。

## 四、超出 PRD 的附加功能（Extras）

以下功能不在 PRD 核心模块中，但已上线并显著提升产品价值：

1. **故事地图（Map）**：`components/views/MapView.tsx`、`features/map/api.ts` — 地点节点、自定义图标、地点连线、角色足迹。
2. **视觉小说编辑器（VN）**：`components/views/VnView.tsx`、`services/vn.rs` — 场景、台词、分支选项、预览播放器。
3. **小说写作模块（Novel）**：`components/views/NovelView.tsx` — 章节列表 + TipTap 长文本编辑器、字数统计。
4. **AI 助手面板**：`components/layout/AiAssistantPanel.tsx`、`features/ai/contextCollector.ts` — 跨功能上下文、流式输出、多厂商支持。
5. **番茄钟**：`components/ui/PomodoroTimer.tsx`、`stores/pomodoro.ts` — 可拖拽、最小化、多主题、音效、每日专注记录。
6. **MC 全局主题与像素字体**：`styles/themes.css` 提供 `mc` 主题，番茄钟/图标/音效均为原创像素风格。
7. **自定义字体导入**：`features/font/api.ts` 支持 `.ttf/.otf/.woff/.woff2` 导入。
8. **时间轴多视图模式**：`TimelineView.tsx` 除标准时间轴外，提供甘特图、树状图、文本模式。
9. **关系矩阵**：`components/views/RelationshipMatrix.tsx`。
10. **高级统计图表**：思维导图、脑状关系网、大纲树图、角色弧线图、情节密度图。
11. **事件图片附件**：`TimelineView.tsx:1869-1901` 支持上传/删除图片。
12. **时间轴右键菜单**：`components/ui/ContextMenu.tsx` 提供编辑/删除/复制/改状态/起连接/问 AI 等。
13. **一致性冲突可视化**：冲突事件卡片红色角标/边框，编辑弹窗顶部提示。

## 六、v2.6.0 建议方向

1. **统一富文本与自动保存**
   - 将事件编辑弹窗、大纲编辑弹窗、角色描述等 `Textarea` 替换为 `RichEditor`；扩展 RichEditor 图片上传。
   - 引入全局 draft 状态与防抖自动保存，保留“保存中/已保存”状态指示。

2. **大纲体验升级**
   - 接入 `@dnd-kit` 或类似库实现大纲树拖拽排序与层级调整。
   - 提供内置大纲模板（三幕式、英雄之旅等）并支持保存自定义模板。
   - 增加多选节点后的批量改状态/删除。

3. **资料库完整化**
   - 左侧显示文件夹树，支持新建文件夹、拖拽归档。
   - 增加“全局笔记 / 工作区笔记”切换；笔记内支持图片嵌入。

4. **写作目标与番茄钟联动**
   - 设置中新增每日字数目标；在番茄钟、Novel 视图、统计视图中展示进度。

5. **可访问性补强**
   - 增加高对比度主题；补齐所有自定义控件的 `aria-label`/`role`。
   - 为时间轴、大纲提供键盘导航与快捷键（如 ↑/↓ 选择节点、Enter 编辑）。

6. **统计视图补全**
   - 后端增加字数统计服务（按章节/角色/事件）；前端新增字数卡片与时间跨度总览。

7. **角色与头像**
   - 角色卡片支持头像图片上传/裁剪；新增“角色时间轴出场分布”热力图或筛选视图。
