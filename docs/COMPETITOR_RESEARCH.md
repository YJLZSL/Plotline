## 三、各产品深度分析

### 3.1 Scrivener
**优势**：
- 成熟的写作 IDE，软木板/大纲/编译系统久经考验。
- 支持 20+ 导出格式（ePub、PDF、Word、Final Draft 剧本格式）。
- 文件级本地存储，作者完全掌控数据。

**劣势**：
- 无时间轴/角色关系可视化（依赖软木板手动排布）。
- 无 AI 集成。
- 界面陈旧，学习曲线陡峭，无中文原生支持。
- 无自动更新，版本迭代慢（v3 发布于 2017，多年后才更新）。

**Plotline 可借鉴**：编译导出系统（多格式）、研究文件夹组织方式。

### 3.2 Plottr
**优势**：
- 时间线 + 大纲是核心卖点，交互流畅。
- 角色弧线模板内置（英雄之旅、三幕式等）。
- 跨平台同步。

**劣势**：
- 纯云端，本地优先作者不信任。
- 订阅制，长期成本高。
- 无 AI、无 VN 脚本、无地图。
- 可视化仅限时间线，无统计图表。

**Plotline 可借鉴**：故事模板（英雄之旅等——Plotline 已有 `seed_template`）、时间线拖拽流畅度。

### 3.3 Campfire Write
**优势**：
- 模块化设计（角色、种族、物种、地点、地图、宗教等），世界观管理强大。
- 地图模块支持在地图上标注地点。
- UI 现代，交互友好。

**劣势**：
- 纯云端 + 订阅制，数据不在用户手中。
- 无时间轴、无 AI、无 VN。
- 导出受限，锁定风险高。

**Plotline 可借鉴**：世界观模块（种族/物种/宗教等扩展）、地图模块交互（Plotline 已有基础地图）。

### 3.4 Obsidian + 插件
**优势**：
- 本地 Markdown 文件，数据完全掌控。
- 插件生态丰富（Full Calendar、Kanban、Excalidraw 等可近似时间线/关系图）。
- 免费 + 离线，同步可选付费。
- AI 插件（Smart Connections、Copilot）支持本地模型。

**劣势**：
- 需要大量配置才能达到 Plotline 的开箱即用体验。
- 无原生时间轴/角色关系图/VN 脚本/番茄钟。
- 可视化能力受限于插件质量。

**Plotline 可借鉴**：插件化思路（未来可开放扩展）、Markdown 互操作、本地模型支持（Plotline 已支持 Ollama）。

### 3.5 Notion
**优势**：
- 数据库 + 文档一体化，灵活度高。
- AI 写作助手内置（Notion AI）。
- 协作能力强。

**劣势**：
- 纯云端，本地优先缺失。
- 无专门的叙事可视化（时间轴/关系图/地图均无）。
- 无 VN 脚本、无番茄钟。
- 订阅制，长期成本高。

**Plotline 可借鉴**：AI 写作交互模式（行内续写、摘要生成）。

## 四、Plotline 的差异化优势

1. **本地优先 + SQLite**：数据完全在用户手中，离线可用，无订阅。这是对 Plottr/Campfire/Notion 的核心差异。
2. **叙事可视化全家桶**：时间轴 + 角色关系 + 大纲树 + 故事地图 + VN 脚本 + 统计图表，目前无单一竞品能覆盖全部。
3. **VN 脚本 + Ren'Py 导出**：这是 Plotline 独有功能，面向游戏叙事设计师的细分市场。
4. **AI 多服务商 + RAG**：支持 OpenAI/硅基流动/火山方舟/腾讯混元/DeepSeek/Moonshot/智谱/Ollama 共 9 种预设，竞品中最多；RAG 检索增强让 AI 基于工作区内容回答。
5. **中文原生**：界面、文档、社区全中文，对中文创作者零门槛。
6. **免费开源**：MIT 许可，无锁定风险。

## 五、Plotline 的劣势与改进方向

| 劣势 | 改进方向 | 优先级 |
|---|---|---|
| 导出格式少（仅 JSON/MD/Ren'Py） | 增加 ePub、PDF、Word 导出 | P1 |
| 无云端同步 | 可选的端到端加密同步（保持本地优先） | P2 |
| 无协作 | 多人评论/批注（v2.0 考虑） | P2 |
| AI 无流式输出 | 支持 SSE 流式响应 | P1 |
| 无移动端 | 响应式 Web 或 Tauri 移动版 | P2 |
| 世界观模块薄 | 参考 Campfire 增加种族/物种/宗教等实体 | P1 |
| 无编译模板 | 参考 Scrivener 的编译系统 | P2 |

---

## 六、总结

Plotline 在**本地优先 + 叙事可视化 + VN 脚本 + AI 多服务商**四个维度具有独特竞争力，
当前定位精准。下一步应聚焦：

1. **巩固优势**：完善 VN 编辑器（富文本、立绘插槽、预览播放器）。
2. **补齐短板**：增加导出格式（ePub/PDF）、AI 流式输出、世界观实体扩展。
3. **体验打磨**：空状态插画、主题预设、性能与可访问性审计。
