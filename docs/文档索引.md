## 按场景查找文档

### 新 AI 接手（第一次进入项目）
1. 先读 `AGENTS.md` — 了解编码规范和禁忌
2. 再读 `交接文档.md` — 了解当前状态和待办
3. 然后读 `产品需求与设计文档.md` — 了解需求事实源
4. 最后读 `docs/密钥管理指南.md` — 了解密钥位置和配置方法

### 需要发布新版本
1. 先读 `docs/密钥管理指南.md` — 确认密钥配置
2. 读 `docs/插件说明.md` — 了解发布流程
3. 读 `交接文档.md` — 了解当前版本状态
4. 执行 `AGENTS.md` 中的提交检查清单

### 修改数据库结构
1. 读 `docs/数据模型.md` — 了解现有模型
2. 读 `docs/技术决策.md` ADR-002 — 了解 SQLite 决策
3. 在 `src-tauri/migrations/` 新增迁移文件（只新增，不修改旧文件）
4. 同步更新 `src/types/` 和 `src-tauri/src/models/`

### 修改 IPC 接口
1. 读 `docs/架构设计.md` — 了解前后端通信
2. 修改 `src-tauri/src/commands/` + `src-tauri/src/services/`
3. 同步 `src/lib/ipc.ts` + `src/features/<domain>/api.ts`
4. 同步 `src/types/`

### 修改主题/样式
1. 读 `src/styles/themes.css` — 了解主题变量
2. 读 `AGENTS.md` §6 — 美术与交互规范
3. 修改时遵循 Tailwind 类名规范，禁止写裸 CSS
