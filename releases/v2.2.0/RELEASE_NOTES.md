# Plotline v2.2.0 Release Notes

## 🎯 版本目标
基于用户反馈，重新打磨时间轴/事件核心逻辑与动画性能，深度重构 MC 主题与番茄钟体验，改进工作区保存逻辑，并支持自定义字体导入（默认内置得意黑/Smiley Sans）。

---

## ✨ 新增功能

### 1. 时间轴/事件逻辑重写
- **真实日历映射**：新增 `timeScale.ts`，使用 UTC 小时/天/月/年日历步进，替换原先固定 30/365 天的近似计算。
- **刻度严格对齐**：`DateRuler` 主/次刻度与事件坐标完全对齐，消除时间轴错位。
- **真实横向拖拽**：移除 `EventCard.dragConstraints`，支持无约束横向拖拽；绝对事件拖拽直接写入真实日期，相对事件按最终位置重新计算 `sortOrder`。
- **连线交互增强**：连线层支持 hover 加粗并点击删除连接。
- **冲突可视化**：一致性冲突事件在卡片上显示红色角标/边框，编辑对话框顶部显示冲突提示。
- **颜色选择器**：事件编辑对话框新增马卡龙色板颜色选择器。
- **甘特图升级**：改用真实日期布局，相对事件排在左侧按 `sortOrder` 堆叠。
- **性能优化**：轨道事件卡片引入水平虚拟化，连线层使用 `useMemo` 缓存可见路径。

### 2. 工作区保存改进
- **元数据编辑**：工作区选择器支持重命名、编辑描述与封面颜色。
- **定时自动备份**：新增后台备份调度器，按 `app_settings.auto_backup` 与 `backup_interval_hours` 定期备份数据库。
- **撤销/重做扩展**：undo/redo 覆盖工作区元数据编辑。
- **Bundle 版本校验**：导入工作区时校验版本号（仅支持版本 2），防止旧格式数据损坏。
- **时间戳联动**：数据库触发器自动将子表变更（事件、角色、笔记等）联动更新 `workspaces.updated_at`。

### 3. 自定义字体导入
- **默认字体**：内置得意黑（Smiley Sans）作为默认字体，通过 `@font-face` 动态注入。
- **导入支持**：设置面板支持导入 `.ttf/.otf/.woff/.woff2` 到 `app_data/fonts/` 并即时生效。
- **模型统一**：`fontTheme` 与 `uiFont`/`editorFont` 统一为预设/自定义模型，避免互相覆盖。

---

## 🔧 改进与优化

- **动画提速**：全局动画时长统一压缩至 150–200ms，更快响应。
- **无障碍支持**：新增 `prefers-reduced-motion` 媒体查询，尊重系统减少动效偏好。
- **微交互反馈**：用 CSS `active:scale` 替代部分 Framer Motion `whileTap`，减少 JS 动画开销。
- **保存状态保护**：新增全局保存状态指示 `SaveStatus` 与 `BeforeUnloadGuard`，防止未保存关闭时丢失数据。
- **MC 主题再设计**：
  - 引入泥土棕/草绿/红石红/圆石灰/木板褐配色，避免单一"绿油油"色调。
  - 控件方块化：按钮、输入框、文本框、选择框圆角降至 2px。
  - 番茄钟音效闭环：开始/暂停/重置/阶段切换/完成均播放 8-bit 音效；MC 主题完成时播放 explosion 音效。
  - 修复设置预览与 accent 冲突：MC 主题固定草绿强调色，用户自定义 accent 不再覆盖。

---

## 🐛 修复

- 修复时间轴 `timeToX` 固定 30/365 天近似导致的刻度错位。
- 修复事件卡片拖拽 `dragConstraints` 限制导致无法真实横向移动的问题。
- 修复相邻事件拖拽后 `sortOrder` 估算不准确的问题。
- 修复 MC 主题下设置预览为空、accent 色冲突的问题。
- 修复 `src/lib/sound.ts` 中 `as unknown as` 类型断言违规。
