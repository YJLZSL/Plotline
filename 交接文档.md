## 已完成工作（v2.7.0 本轮迭代）

### 1. Markdown 渲染修复
- **需求**：AI 面板截图显示 `**`、`*` 等 Markdown 语法被转义为纯文本，影响可读性。
- **实现**：
  - 新增 `src/components/ui/Markdown.tsx`，封装 `react-markdown` + `remark-breaks` + `rehype-sanitize`。
  - `AiAssistantPanel.tsx` 对助手消息统一使用 `<Markdown />` 渲染。
  - 在 `promptTemplates.tsx` 与 AI 建议词生成处统一文本清洗，禁止裸星号输出。
- **状态**：✅ 已完成，新增 Markdown 渲染与清洗单元测试

### 2. 动画性能与视觉打磨
- **需求**：界面动画拖沓、存在冗余 `layout` 重排，部分视图切换掉帧；字体缺失时回退不正确。
- **实现**：
  - 全局动画时长统一为 150–250ms，缓动统一为 `[0.16, 1, 0.3, 1]`。
  - 核心视图切换迁移到 `transform`/`opacity` GPU 加速，移除冗余 `layout` 重排。
  - 新增“炫技微动画”开关与可选粒子/波纹效果，默认关闭并尊重减少动画偏好。
  - 修复 sans/mono/pixel/得意黑字体回退栈。
- **状态**：✅ 已完成，Playwright 录制关键视图切换帧时间有改善

### 3. 时间轴交互与 UI 重构
- **需求**：时间轴事件卡片/画布可拖拽到最左侧继续溢出，轨道头部与筛选栏对齐松散，缺少明确的入场/退场动画。
- **实现**：
  - `src/features/timeline/timelineLayout.ts` 新增 `computeEventDragConstraints()` 与 `clampTimelineScroll()`，限制事件卡片与画布左边界。
  - 优化轨道头部、事件卡片、筛选栏间距与对齐。
  - 视图/折叠/筛选增加明确的 `AnimatePresence` 入场/退场动画。
  - 新增 `TimelineEmptyIllustration` 空状态插画。
- **状态**：✅ 已完成，新增时间轴拖拽边界与动画单元/E2E 测试

### 4. AI 助手能力明确化与架构升级
- **需求**：用户不清楚 AI 能做什么，缺少一键触发的场景化能力；上下文利用不足，重复请求成本高。
- **实现**：
  - `AiAssistantPanel.tsx` 顶部新增“能力快捷栏”：优化当前事件、优化当前时间轴片段、总结整个工作区、检查时间轴逻辑漏洞。
  - 扩展 `AiChatContext` 支持“当前选中实体 / 当前视图 / 整个工作区”三种范围。
  - 后端新增 `ai_cache` 与 `ai_conversation_history` 表实现 KV Cache。
  - 新增关键词 RAG 检索，按角色/地点/事件/大纲组装 prompt。
  - 新增 `ai_prompts.rs` 与四个对应 Tauri 命令。
- **状态**：✅ 已完成，新增 Rust 与前端单元测试覆盖 RAG、KV Cache、快捷操作

### 5. 番茄钟与 MC 主题趣味化
- **需求**：番茄钟成就无法重置，完成时缺少庆祝反馈；MC 主题彩蛋不足。
- **实现**：
  - `pomodoro.ts` 新增 `resetAchievements` action。
  - `PomodoroTimer.tsx` 新增“重置成就”按钮与完成庆祝动画（粒子爆裂 + 奖杯/稀有矿石弹窗）。
  - MC 主题增加 Creeper 惊喜彩蛋与稀有矿石随机掉落。
  - 改进像素方块高光/阴影质感与 8-bit 音效反馈。
- **状态**：✅ 已完成，新增番茄钟重置与 MC 主题单元/E2E 测试
