## 2. v2.3.0 待完成重构任务（优先级）

### Phase 1 — 数据模型与后端扩展
- 为 `Event` 增加可选 `locationId` 字段，关联 `locations`。
- 新增迁移 `src-tauri/migrations/008_event_location.sql`。
- 更新 Rust `models/event.rs`、`services/event.rs`、`commands/event.rs`。
- 同步前端类型 `src/types/event.ts`。

### Phase 2 — 主界面横向可滑动
- 重构 `src/components/views/WorkspaceSelector.tsx` 为横向轮播/滚动。
- 卡片展示封面色块/图片、名称、描述、事件数量。
- 支持鼠标/触控板横向滚动，保留新建/导入/设置入口。

### Phase 3 — 时间轴增强与修复
- **修复关联线**：使用 DOM 测量（`ResizeObserver` + `requestAnimationFrame`）绘制从卡片边缘到边缘的连线；拖拽时实时更新。
- **树状时间轴**：新增 `src/features/timeline/treeLayout.ts` 与 `TreeTimeline.tsx`，在 `TimelineView` 加入 `tree` 视图。
- **文本模式**：新增 `TextTimeline.tsx`，支持列表编辑与“只写正文”详情侧栏。
- **事件详情增强**：`EventEditDialog` 新增地点字段、人物头像显示、富文本描述。

### Phase 4 — MC 主题重构
- 调研 Minecraft 官方 UI 配色（泥土/草方块/圆石/木板/红石/钻石）。
- 重绘 `src/styles/themes.css` `[data-theme="mc"]` 背景、控件、纹理（CSS/SVG，避免版权素材）。
- 调整对比度确保可读性。
- 更新 `PomodoroTimer` MC 主题进度条与音效。

### Phase 5 — 番茄钟与创作联动
- 扩展 `src/stores/pomodoro.ts`：支持绑定事件/大纲节点，记录每日专注时长与会话历史。
- 在时间轴、大纲、笔记工具栏增加“开始专注”入口。
- 番茄钟悬浮窗显示绑定目标与今日专注统计。
- 统计页新增番茄钟图表。

### Phase 6 — 教程系统
- 在 `SettingsView` 新增 `help` 页签。
- 新增 `src/features/help/tutorialData.ts`，定义时间轴/角色/大纲/地图/VN/番茄钟/AI/设置教程。
- 支持分步导航与首次进入工作区引导气泡。

### Phase 7 — 测试、文档与发布
- 单元测试：`treeLayout.test.ts`、关联线回归测试、`pomodoro` 联动测试。
- E2E：主界面滑动、树状时间轴、MC 主题、番茄钟绑定。
- 更新 `docs/文档索引.md` 与 `交接文档.md`。
- 版本号升级到 `2.3.0`，更新 `更新日志.md`。
- 构建签名：`pnpm build:tauri` + `pnpm tauri signer sign ...`
- 生成 `releases/v2.3.0/latest.json`。
- `git tag v2.3.0` 并推送，上传产物，发布 Release。
