## 已完成工作（v2.5.4 本轮迭代）

### 1. 番茄钟可拖动与最小化
- **需求**：番茄钟面板固定在工作区右下角，容易遮挡时间轴主画布，需要自由拖动与快速收起。
- **实现**：
  - `src/stores/pomodoro.ts` 新增 `position` 与 `minimized` 状态，接入 Zustand persist 持久化。
  - `PomodoroTimer.tsx` 面板使用 `motion.div` 包裹，标题栏作为拖拽手柄，约束在视口内。
  - 新增最小化/恢复按钮；最小化后渲染为悬浮芯片，展示剩余时间，点击恢复。
  - `WorkspaceLayout.tsx` 在最小化时渲染芯片，不再占据布局空间。
- **状态**：✅ 已完成，单元测试覆盖拖拽边界与最小化切换

### 2. 时间轴右键菜单
- **需求**：提升时间轴操作效率，减少频繁点击工具栏。
- **实现**：
  - 新增 `src/components/ui/ContextMenu.tsx` 无头组件：基于原生 `contextmenu` 事件 + Portal，支持键盘导航与 Escape 关闭。
  - `TimelineView.tsx` 接入三类右键菜单：
    - 事件卡片：编辑、删除、复制、更改状态、开始连接、问 AI。
    - 轨道行：重命名、更改颜色、隐藏/显示、删除、添加事件。
    - 画布空白区：添加事件（选择轨道）、粘贴事件、放大/缩小、一致性检查。
  - i18n 键同步到 `zh-CN.json` / `en.json`。
- **状态**：✅ 已完成，E2E 测试覆盖三类菜单

### 3. 主题质感升级
- **需求**：主题视觉层次不够细腻，设置页主题选择缺乏直观预览。
- **实现**：
  - `src/styles/themes.css` 新增 `--shadow-card-soft`、`--shadow-elevated-soft`、`--bg-base-gradient`、`--surface-highlight` 等变量。
  - light/sepia/dark 主题背景使用柔和渐变；MC 主题减少文字面纹理覆盖，增强文本阴影/遮罩保证可读性。
  - 卡片、按钮、输入框的 hover/focus ring 统一使用 accent 色。
  - `SettingsView.tsx` 主题卡片新增 live mini-preview，实时展示主题效果。
- **状态**：✅ 已完成，E2E 截图测试覆盖主题切换

### 4. AI 跨功能上下文
- **需求**：AI 助手此前只能拿到当前视图片段，回答片面，需要基于全工作区上下文给出建议。
- **实现**：
  - `src/types/ai.ts` 扩展 `AiChatContext`，覆盖工作区摘要、时间轴、角色、地点、大纲、笔记、选中实体。
  - Rust `AiChatInput` 模型同步 `context` 字段。
  - 新增 `src/features/ai/contextCollector.ts`，汇总多维度工作区数据。
  - `AiAssistantPanel.tsx` 新增上下文来源开关（时间轴/角色/地点/大纲/笔记）。
  - Rust `ai_chat` 服务按 Token 预算截断并组装上下文 Prompt。
  - `useApplyAiOutput` 扩展 `applyTo` 目标，支持写入角色、地点、大纲节点、笔记；后端新增 `apply_ai_output` 对应目标支持。
- **状态**：✅ 已完成，单元测试覆盖上下文组装与 apply-output 映射

### 5. AI 面板两列布局
- **需求**：会话列表与消息区挤在同一面板，操作效率低。
- **实现**：
  - `AiAssistantPanel.tsx` 重构为可拖拽调整宽度的左右两栏：左侧会话列表，右侧消息流 + 输入框。
  - 上下文标签从消息气泡上方移至输入框上方，支持逐个移除。
  - 建议词改为横向滚动列表，点击一键填充并发送。
  - 单条消息操作按钮收敛为按当前视图适用性分组的下拉菜单。
  - 统一颜色 token 为 `bg-bg-surface` / `border-border` / `text-text-primary`。
- **状态**：✅ 已完成，单元测试覆盖布局与建议词交互
