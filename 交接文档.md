## 已完成工作（v2.5.0 本轮迭代）

### 1. 时间轴图片插入
- **需求**："时间轴那些地方加入图片插入的功能"

- **数据库**：迁移 `011_event_images.sql` 给 `events` 添加 `image_urls` JSON 字段
- **状态**：✅ 已完成，测试通过

### 2. 小说写作模块
- **需求**："加个小说模块，这样可以根据时间线和大纲写小说，然后再做成vn"
- **实现**：
  - 全新「小说」视图，左侧章节列表 + 右侧 TipTap 富文本编辑器
  - 支持章节关联大纲节点，从大纲结构直接写小说内容
  - 实时字数统计（中文 + 英文），自动保存
  - 章节状态管理：草稿/已完成/待修改
- **数据库**：迁移 `012_novel_chapters.sql` 创建 `novel_chapters` 表
- **状态**：✅ 已完成，测试通过

### 3. VN 增强
- **需求**："然后再做成vn"（VN 编辑器增强）
- **实现**：
  - 台词新增「立绘位置」选择器：左/中/右
  - 预览面板根据位置渲染立绘
  - 新增「完整预览」对话框，支持分支选择与场景跳转
- **数据库**：迁移 `013_vn_sprites.sql` 给 `vn_lines` 添加 `sprite_position` 字段
- **状态**：✅ 已完成，测试通过

### 4. MC 主题与番茄钟真正 Minecraft 元素
- **需求**："mc主题和番茄钟还是不够凸显，感觉只是像素风，而不是我的世界风，缺少像苦力怕，钻石块，红石块等等这些元素"
- **实现**：
  - 新增 7 种像素纹理：苦力怕脸、钻石矿石、红石块、黑曜石、金块、铁块、绿宝石
  - 番茄钟进度条按完成度使用不同方块类型（泥土→圆石→铁→金→钻石）
  - 100% 完成时触发 Creeper 爆炸动画
  - 成就指示器用方块图标展示已完成专注次数
  - UI 控件纹理：主按钮金块、成功按钮绿宝石、选中标签钻石矿石
- **状态**：✅ 已完成，测试通过

## v2.5.2 迭代状态

### 目标

新增环境动画与减少动画开关，让用户可全局控制界面动效；完成 v2.5.1 的 MC 主题增强后统一发布 v2.5.2。

### 已完成工作

- 后端新增 `app_settings.animations_enabled` 列与模型/服务逻辑，前端类型同步到 `src/types/settings.ts`。
- 新增 `src/stores/motion.ts` 与 `src/hooks/useAmbientAnimation.ts`，提供基于 `animations_enabled` 的门控动画参数。
- `src/stores/ui.ts` 在 `applyToDOM` 中写入 `--motion-enabled` CSS 变量，`src/styles/themes.css` 提供对应辅助类。
- `SettingsView` 外观标签新增「启用动画」切换，i18n 键同步到 `zh-CN.json`/`en.json`。
- `WorkspaceSelector`、`Sidebar`、`TimelineView`、`PomodoroTimer`、`EmptyState`、`SettingsView` 主题卡片等组件接入运动设置，实现悬停/入场/反馈等细腻动画，同时尊重用户减少动画偏好。
- 新增对应单元测试与 E2E 测试，覆盖 motion store、useAmbientAnimation hook、设置开关与核心视图动画门控。

### 发布与签名状态
