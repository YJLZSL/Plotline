## 已完成工作（v2.7.3 本轮迭代）

### 1. Rust 编译错误修复
- **需求**：v2.7.2 tag 推送后，CI/Release workflow 在 `src-tauri/src/services/ai.rs` 等处报 Rust 编译错误，未生成可用 Release。
- **实现**：
  - 修复 `ai.rs` 中 `NeedApi` 字段匹配不完整导致的 exhaustiveness 编译错误。
  - 补全 `UpdateWorkspaceInput` 缺失字段，使其与前后端模型保持一致。
  - 修正 `entities` 相关代码中的借用/移动问题，避免所有权冲突。
- **状态**：✅ 已完成，本地 `cargo check` 与 CI 编译预期全绿

### 2. v2.7.2 功能完整保留

### 1. Markdown 渲染深度修复
- **需求**：v2.7.0 中 AI 回复的 `**`、`*` 等 Markdown 标记仍被原样显示为文本，表格/删除线/GFM 也未正确渲染。
- **实现**：
  - `src/components/ui/Markdown.tsx` 引入 `remark-gfm`，支持表格、删除线、任务列表。
  - 新增 `src/components/ui/markdownUtils.ts`，重构 `normalizeMarkdown`：解码 HTML 实体、归一化全角星号、还原双重转义、合并跨行标记、等待流式标记闭合。
  - 移除 `remark-breaks`，改用 CSS `white-space: pre-wrap`。
- **状态**：✅ 已完成，新增 Markdown 渲染与清洗单元测试

### 2. 动画性能再优化
- **需求**：v2.7.0 动画仍显拖沓，多层 `AnimatePresence mode="wait"` 与 stagger delay 导致视图切换与列表入场迟缓；`will-change-transform` 滥用造成合成层爆炸。
- **实现**：
  - `src/lib/motion.ts` 新增 `MOTION_FAST`（120ms），核心视图统一 120–200ms 动效。
  - 减少 `mode="wait"` 嵌套：`WorkspaceLayout` 保留、`SettingsView` 取消、`TimelineView` 改为更轻量的 crossfade。
  - 压缩 `TrackLane`/`EventCard` stagger delay（最多 50ms，事件卡片统一 0ms）。
  - 移除大范围 `will-change-transform`，仅在动画运行时动态应用。
- **状态**：✅ 已完成，关键视图切换测试覆盖

### 3. 时间轴交互与 UI 全面打磨
- **需求**：v2.7.0 事件卡片/画布仍可拖到最左侧之外；工具栏拥挤、事件卡片样式单一、日期标尺标签重叠、连线视觉噪声大。
- **实现**：
  - `src/features/timeline/timelineLayout.ts` 修正 `computeEventDragConstraints` 为基于目标位置计算约束；同步阻止画布越界滚动并增加 CSS overscroll 控制。
  - `src/components/views/TimelineView.tsx` 重构工具栏与筛选栏层级；事件卡片宽度自适应、主题表面色 + 4px 轨道色条；日期标尺防重叠；连线降噪；“今天”参考线主题化。
- **状态**：✅ 已完成，新增时间轴拖拽边界、宽度计算、标尺防重叠单元/E2E 测试

### 4. AI 助手能力再明确化
- **需求**：v2.7.0 用户仍不清楚 AI 能做什么，默认上下文源过少，RAG/KV Cache 效果不可见。
- **实现**：
  - `AiAssistantPanel.tsx` 新增能力说明折叠面板、`/` 命令菜单、上下文摘要显示（已引用实体数与 RAG 检索数量）。
  - `src/stores/aiContext.ts` 默认启用更多上下文源；`src/features/ai/contextCollector.ts` 提升 `MAX_DESC_LENGTH`/`MAX_ITEMS`，新增“读取整个工作区”模式。
  - 后端 KV Cache 键加入上下文哈希与工作区 `updated_at`，改进中文 2-gram 分词，提升 `MAX_CHUNK_TOKENS` 到 2000。
- **状态**：✅ 已完成，新增 AI 能力面板、`/` 命令、上下文摘要测试

#### 2.5 番茄钟成就系统与 MC 主题修复
- **需求**：v2.7.0 “重置成就”会同时重置计时器；MC 彩蛋缺少开关、完成庆祝动画几乎无法触发。
- **实现**：
  - `src/stores/pomodoro.ts` 拆分 `resetAchievements`（仅重置计数）与 `resetTimer`（重置计时器）。
  - 新增 `achievements: { date, count, streak }`，显示今日专注数与连续天数。
  - 完成时延迟 1.8 秒再切换阶段，确保庆祝/爆炸动画可见。
  - `PomodoroTimer.tsx` 增加 `mcEasterEggsEnabled` 开关，限制 Creeper/sparkle 触发频率。
- **状态**：✅ 已完成，新增番茄钟成就系统、重置拆分、MC 彩蛋开关单元测试

#### 2.6 MC 主题与字体兼容性
- **需求**：v2.7.0 MC 主题全局选择器过宽污染其他组件；非 MC 主题下 `--bg-base-gradient` 未定义导致预览卡片透明；设置页字体预览使用内联 style。
- **实现**：
  - `src/styles/themes.css` 收紧 MC 纹理选择器，仅对 `.mc-button` / `.mc-textured` 应用；补全 `:root`/dark/sepia 的 `--bg-base-gradient`。
  - `src/styles/tailwind.css` 给 `@font-face` 增加 `local()` 源，优先使用本地已安装字体。
  - `src/components/views/SettingsView.tsx` 移除内联字体预览 style，修复导入顺序，tab 动画改用更轻的 opacity/y 微动。
- **状态**：✅ 已完成，新增主题/字体兼容性测试

#### 2.7 调研与规划
- **实现**：
  - 输出 `docs/v2.7.2-research.md`，含截图分析、竞品参考（Notion Timeline、Aeon Timeline、Claude、Kimi/豆包/通义）、中文社区反馈（B 站/知乎/V2EX/NGA/小红书）、动画前端效果建议。
- **状态**：✅ 已完成
