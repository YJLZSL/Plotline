- **需求**："点击时间轴展开详情页，显示具体时间、地点、人物以及详情描述"
- **实现**：事件编辑对话框新增「地点」选择器，关联地图中的地点；后端 `location_id` 已完整支持
- **状态**：✅ 已完成

### 3. MC 主题素材丰富

2. 重新推送 tag 触发 CI（或删除后重新创建 `v2.4.0` tag）

**方案 B：手动签名（无需重新构建）**
1. 从 Release 页面下载 `Plotline_2.4.0_x64-setup.exe`
2. 使用本地密钥签名：`tauri signer sign --private-key-path keys/plotline.key -- Plotline_2.4.0_x64-setup.exe`
3. 生成 `.sig` 文件和 `latest.json`，上传到 Release

**方案 C：移除签名验证（不推荐）**
- 修改 `tauri.conf.json` 移除 `pubkey` 字段，但会降低安全性
